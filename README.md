# Jamba-GSheets
This repository hold code for custom function in G sheets that calls Jamba Completion API 

How to install :
1. Open new google sheet
2. Extension -> Apps scripts
3. Create new script file
4. Paste the code from Jamba_Chat file, and insert your API-KEY the replace the XXX part -   Authorization: 'Bearer XXX'
5. Call the function "Jamba_Chat" within a cell
6. Function's parameters are (prompt, system, temperature=0). you can control all of them
